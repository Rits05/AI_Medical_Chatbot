###🏥 AI Medical Chatbot with RAG and Medical Report Analysis
**Project Overview**

Developed an AI-powered Medical Chatbot that combines Retrieval-Augmented Generation (RAG) with Medical Report Analysis to assist users in understanding medical documents, laboratory reports, and medicine information. The chatbot answers questions from uploaded medical documents, analyzes laboratory test reports, and generates easy-to-understand health summaries using Large Language Models (LLMs).

**Objectives**
-Build an AI chatbot for medical question answering.
-Analyze laboratory reports automatically.
-Extract laboratory test values from different report formats.
Provide educational explanations of abnormal findings.
Support multiple medical report types.
Generate professional AI-based report summaries.
